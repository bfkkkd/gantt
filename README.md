base on jQueryGantt
===========
Try the online working demo here: https://gantt.twproject.com

功能：

搜索筛选：
1.支持关键字搜索。
2.支持只显示未完成项目。
3.回车直接提交。

数据管理：
1.冲突管理，加入版本被修改提示，限制保存。
2.权限管理，加入简单权限管理，保存IP记录
3.加入每天备份机制，意外情况可恢复上一天修改情况。

便捷功能
1.状态自动变更，项目名称后填入 空格+开发 或 空格+测试 或 空格+联调 自动识别颜色。

BUG
1.修复资源列表无法减员问题。
2.修复错误提示报错问题。
3.修正未登录用户场景，项目时间展示不变灰问题。
